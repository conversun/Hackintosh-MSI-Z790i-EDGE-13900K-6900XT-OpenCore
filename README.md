# Hackintosh-MSI-Z790i-EDGE-13900K-6900XT-OpenCore


## Hardware

* Mainboard: MSI Z790i EDGE
* CPU: i9 13900KF
* RAM: 32G DDR5 * 2 (OC 7000Mhz)
* Boot SSD: SN850X
* Graphics: 6900XT
* WIFI/Bluetooth: BCM94360Z3


## BIOS

* Enable D.T.M
